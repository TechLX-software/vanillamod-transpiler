Introduction

Introduction#

What is vMod?#

vMod is the open source library that powers the vanillamod.com website. vMod translates JavaScript into Minecraft commands (like /gamemode creative). The result is the practicality of a real programming language, while leveraging the features and compatibility of Minecraftâ€™s built in features. vMod only works for Minecraft Java Edition.

What is vanillamod.com?#

vanillamod.com is the website we (TechLX) built to help teach programming and game design. On the website, you can save mods to your account and learn with free and paid courses. But, it's also powerful enough for true map-making, with new features added frequently.

What is this documentation for?#

This documentation describes the features of the vMod library and the code editor on vanillamod.com. It is primarily a reference for writing JavaScript using the vMod library.

Want to start making things ASAP?

The best tutorial is the "Intro to Modding (with Minecraft)" eLearning course on vanillamod.com.
Sign in, go to the courses page, then click "Enroll".

If you're already familiar with Minecraft and programming, try the Quick Start guide on the next page.

What can I do with vMod?#

Stuff - (Create games for vanilla MC, Learn JS, Learn game development)

Check out what others have made on the Examples page.

Why should I use vMod?#

Stuff - (As a Student, Educator, Game Developer / Map Maker.) (Learn JS, online courses and documentation, Convenience and power of real code syntax; no installs or configuraton, just add the MC world to your saves; helpful editor checks for command errors; performance optimization and analyzer)

The Editor page has a more detailed explanation of many helpful features.

How does vMod work?#

Stuff - (vMod looks over code, turns it into MC commands, combines into a Minecraft datapack that can be easily added to your world.)

A more detailed and technical explanation be found on the Transpiler page.
